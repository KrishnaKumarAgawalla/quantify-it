# Quantify.it 📈
**The Systematic Investment Planner for the Modern Investor.**

Quantify.it helps investors split a lump sum amount into a custom ratio of assets (ETFs/Stocks) and generates a precise daily, monthly, or yearly buying schedule to maintain that ratio over time.

## 🚀 Features
- **Smart Ratio Allocation:** Divide capital based on custom weights (e.g., 7:5).
- **Fractional Unit Handling:** A proprietary "Bucket Algorithm" that manages whole-share purchases over time.
- **Dynamic Assets:** Add or remove as many ETFs/Stocks as you need.
- **Mobile Responsive:** Built with Tailwind CSS for a seamless dashboard experience.

## 🛠️ Tech Stack
- **Framework:** React + Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Deployment:** Vercel

## 📦 Installation
1. `npm install`
2. `npm run dev`
